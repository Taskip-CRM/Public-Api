# Meeting Management API

The Meeting Management API provides comprehensive meeting scheduling, booking, and management capabilities. This powerful API allows you to create meetings, manage bookings, handle deal-specific meetings, and integrate with calendar systems for seamless scheduling workflows.

## Overview

Meeting Management consists of multiple interconnected features:
- **Meeting Creation & Scheduling**: Create and schedule meetings with participants
- **Booking System**: Allow clients to book available time slots
- **Deal Integration**: Associate meetings with sales pipeline deals
- **Calendar Integration**: Sync with popular calendar systems
- **Participant Management**: Handle attendees and notifications
- **Meeting Status**: Track completion and outcomes
- **Bulk Operations**: Manage multiple meetings efficiently

## Main Endpoints Overview

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/public-v1/meeting-list` | Get all meetings |
| `GET` | `/api/public-v1/meeting/details/{uuid}` | Get meeting details |
| `GET` | `/api/public-v1/meeting/{uuid}/booking-list` | Get booking list for a meeting |
| `GET` | `/api/public-v1/meeting/{uuid}/booking/details/{id}` | Get specific booking details |
| `POST` | `/api/public-v1/meeting/{uuid}/booking` | Book a meeting slot |

## Deal-Specific Meeting Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/public-v1/sales-pipeline/{uuid}/deal/{id}/meeting` | Get all meetings for a deal |
| `POST` | `/api/public-v1/sales-pipeline/{uuid}/deal/{id}/meeting` | Create a meeting for a deal |
| `GET` | `/api/public-v1/sales-pipeline/{uuid}/deal/{id}/meeting/{meeting_id}` | Get specific deal meeting |
| `PUT` | `/api/public-v1/sales-pipeline/{uuid}/deal/{id}/meeting/{meeting_id}` | Update a deal meeting |
| `DELETE` | `/api/public-v1/sales-pipeline/{uuid}/deal/{id}/meeting/{meeting_id}` | Delete a deal meeting |
| `POST` | `/api/public-v1/sales-pipeline/{uuid}/deal/{id}/meeting/bulk-actions` | Bulk meeting actions |
| `POST` | `/api/public-v1/sales-pipeline/{uuid}/deal/{id}/meeting/{meeting_id}/complete` | Mark meeting as complete |

## Get All Meetings

Retrieve a list of all meetings in your system.

```http
GET /api/public-v1/meeting-list
```

### Request Headers
```http
X-Secret-Key: your-secret-key-here
Content-Type: application/json
```

### Query Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `page` | integer | No | Page number for pagination (default: 1) |
| `limit` | integer | No | Number of meetings per page (default: 50) |
| `status` | string | No | Filter by status (`upcoming`, `completed`, `cancelled`) |
| `type` | string | No | Filter by type (`sales`, `support`, `general`) |
| `participant` | string | No | Filter by participant email |
| `date_from` | date | No | Filter meetings from date (YYYY-MM-DD) |
| `date_to` | date | No | Filter meetings to date (YYYY-MM-DD) |
| `sort` | string | No | Sort field (`start_time`, `created_at`, `title`) |
| `order` | string | No | Sort order (`asc` or `desc`) |

## Book a Meeting

Create a new booking for an available meeting slot.

```http
POST /api/public-v1/meeting/{uuid}/booking
```

### Path Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `uuid` | string | Yes | Meeting UUID |

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `attendee_name` | string | Yes | Attendee full name |
| `attendee_email` | string | Yes | Attendee email address |
| `attendee_phone` | string | No | Attendee phone number |
| `company` | string | No | Attendee company name |
| `position` | string | No | Attendee position/title |
| `booked_time` | datetime | Yes | Preferred meeting time (ISO 8601) |
| `timezone` | string | No | Attendee timezone (default: UTC) |
| `meeting_purpose` | string | No | Purpose of the meeting |
| `message` | string | No | Additional message or requirements |
| `custom_fields` | object | No | Custom form fields data |
| `send_confirmation` | boolean | No | Send confirmation email (default: true) |
| `language` | string | No | Preferred language for communications |
